** 项目名称：**蛇影行踪

项目简介：蛇影行踪是一款基于Spring Boot和Vue构建的Web游戏应用，采用前后端分离的开发模式。
该平台支持用户注册登录、自动匹配、在线对战、Bot代码管理、对战记录、比赛回放、积分与排行榜等功能。
此外，平台还支持多对玩家手动或自定义Bot代码进行人人、人机、机器间对战。
项目遵循微服务架构，分为主服务、匹配服务和Bot服务三个独立的Spring Boot进程。

** 技术栈： **

后端：Spring Boot、Spring Security、MyBatis Plus、Redis、JWT、Lombok
前端：Vue3、Vue Router、Vuex
数据库：MySQL
其他：WebSocket 

** 项目亮点： **
- 游戏地图在每局对战前动态生成，使用Flood Fill算法保证地图连通，确保每场比赛的独特性和挑战性
- 匹配机制通过动态调整匹配条件，确保玩家能够在合理的等待时间内找到对手
- 对局回放机制允许玩家查看历史对局，方便玩家对历史对局进行复盘
- Bot自动战斗机制使玩家能够通过编写自定义代码，控制蛇进行自动对战，提升游戏的可玩性
